Summary: Census vs. Zillow

The purpose of this project was to help our client determine which real estate markets to invest in.

We gathered data from US Census API and Zillow to help determine which census data point is most correlated with average per square foot price. We also looked at poverty levels and percentage increase in per square foot price over time. Across all the data points we looked at, Washington State and Ohio appear to have the greatest potential returns in real estate investments.
